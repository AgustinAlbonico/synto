# Restricciones Técnicas

- **Stack obligatorio**: Python (Flask)
- **Frontend**: HTML + CSS (Tailwind CDN)
- **Base de datos**: SQLite
- **Hosting**: Render free tier
- **Tiempo de carga objetivo**: < 2s
- **Navegadores soportados**: Chrome, Firefox, Safari, Edge (últimas 2 versiones)
- **Responsive**: Mobile-first
- **No usar**: JavaScript frameworks (React/Vue/Angular) para mantener simplicidad
